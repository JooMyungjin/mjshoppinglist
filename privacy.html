개인정보처리방침

최종 업데이트: 2026년 4월 7일

요약: 이 확장프로그램은 개발자에게 어떠한 데이터도 전송하지 않습니다. 모든 데이터는 사용자의 브라우저 내에만 저장됩니다.

1. 수집하는 정보

쇼핑 내역 정리 확장프로그램은 개발자가 수집하는 개인정보가 없습니다.

확장프로그램이 수집하는 정보는 사용자 본인의 쇼핑몰 주문 내역(날짜, 상품명, 금액, 주문번호)이며, 이는 사용자 브라우저의 로컬 스토리지에만 저장됩니다.

2. 데이터 저장 위치

  • 수집된 데이터는 사용자의 브라우저(chrome.storage.local)에만 저장됩니다.
  • 개발자 서버나 외부 서버로 전송되지 않습니다.
  • 구글 시트 연동 기능을 사용하는 경우, 데이터는 사용자가 직접 생성한 본인의 Google 스프레드시트로만 전송됩니다.

3. 외부 서비스 사용

  • Frankfurter API (frankfurter.app): 알리익스프레스 달러 상품의 원화 환율 조회 시 사용됩니다. 요청 시 날짜 정보만 전달되며 개인정보는 포함되지 않습니다.
  • Google Apps Script: 사용자가 직접 설정한 경우에만 사용되며, 사용자 본인의 Google 계정과 스프레드시트로만 데이터가 전송됩니다.

4. 데이터 삭제

확장프로그램 설정 패널의 데이터 삭제 기능 또는 Chrome 브라우저의 확장프로그램 데이터 삭제 기능을 통해 언제든지 모든 데이터를 삭제할 수 있습니다.

5. 문의

개인정보처리방침에 관한 문의는 GitHub Issues를 통해 남겨주세요.
